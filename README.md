# Planty
C'est un Projet WordPress dans le cadre d'une formation de développpeur WordPress. Il sert à mettre en application les connaissances apprises en HTML, CSS et PHP.

### Pré-requis
* HTML 5
* CSS 3
* PHP

### Installation
Le projet est fait en local.
